# Linear Regression Project

## Objective
Predict target variable using linear regression.

## Tools & Libraries
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook, Anaconda

## Dataset
- File: ozeone.csv
- Description: contains the predictive variables and the target variable

## Steps
1. Load CSV data
2. Remove unnecessary columns
3. Check and handle missing values
4. Split data into features (X) and target (y)
5. Standardize features
6. Split data into training and test sets
7. Train Linear Regression model
8. Predict on training set, calculate RMSE & R2
9. Predict on test set, calculate RMSE & R2
10. Display model coefficients
